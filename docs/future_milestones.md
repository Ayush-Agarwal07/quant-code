# Future Milestones

## Milestone 1: Agentic Research Layer

Current. Produce structured, data-aware research packets with deterministic mock behavior,
critiques, experiment plans, backtest stubs, and memory-write proposals.

## Milestone 2: Deterministic Strategy DSL and Backtester

Execute validated `StrategySpec` rules against point-in-time data. Add universe history,
feature computation, transaction costs, robustness analysis, and reproducible result
artifacts.

## Milestone 3: Outcome-Grounded Memory

Persist strategies, failures, research lessons, mutations, and agent calibration only when
linked to actual experiment evidence.

## Milestone 4: Richer Data Connectors

Add versioned connectors for market, event, fundamental, analyst, options, and alternative
data with explicit timestamp and quality guarantees.

## Milestone 5: Paper Trading and Broker Adapter

Introduce a separately reviewed execution boundary after deterministic research and
backtesting are mature. No broker behavior is implemented in the current project.
